# Sales Performance Dashboard (Power BI)

## Overview
Built an end-to-end interactive sales analytics dashboard in Power BI to analyze retail performance across regions and product categories.

## Highlights
- Executive KPIs: Total Sales, Total Profit, Total Orders, Average Order Value
- Interactive slicers and drill-down exploration by region and category
- Visual breakdowns: Sales by Region, Category, Profit by Sub-Category, Top Products
